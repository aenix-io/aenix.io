## Frequently asked questions

### What is Ænix Platform?

A turnkey commercial cloud-in-a-box from Aenix, built on the open-source Cozystack project we created and maintain. It adds the layers a real cloud operator needs — hosting panel, billing, portals, payments, support — and ships in five editions matched to buyer profile.

### What's the relationship between Aenix, Ænix Platform, and Cozystack?

Aenix = the open-core company. Ænix Platform = our commercial productized cloud-in-a-box (5 editions). Cozystack = the open-source CNCF project we created and maintain (Apache 2.0). The separation matters because Cozystack stays vendor-neutral on the CNCF side; Ænix Platform is where commercial value-add lives.

### Is Cozystack free?

Yes. Cozystack is Apache 2.0. Anyone can run it. Aenix is the original maintainer — we're often consulted by self-running organizations. Ænix Platform is for organizations that want a turnkey cloud (billing, portals, payments, support) without building it in-house.

### How big are typical Aenix engagements?

ISP Edition entry from €1.3k/month support tier. Platform Readiness Assessment €20-50k fixed-price (5-10 days). Pilot €100-300k (3-6 months). Full Public Cloud / Enterprise build €500k - €10M+ (9-18 months). Managed operations: monthly retainer post-deployment.

### Who are Aenix's customers?

Public production customers running Ænix Platform ISP Edition: GoHost.kz, HDReady, Beby Cloud, HiKube, UseTech, Cloupard, Cloudsy. We also work with tier-1 European banks under NDA on multi-million-euro Enterprise Edition engagements; first named bank case studies expected mid-2027 as NDAs expire. Sovereign cloud and AI/ML deployments are also active under NDA.

### Why five editions?

Same product, different feature bundles per buyer profile. Public Cloud Edition for large operators; ISP Edition for hosting providers; Enterprise Edition for regulated enterprises; IDP Edition for product engineering teams; AI/ML Edition for AI-heavy organizations. Editions share the Cozystack foundation; bundles differ in portals, billing surfaces, integrations, support tier, and edition-specific modules.

### Can I just run Cozystack myself?

Yes. Cozystack is Apache 2.0; if you have platform engineering capacity, run it directly. Aenix's commercial path (Ænix Platform) is for organizations that want the turnkey package — billing, portals, payments, productized installer, enterprise support — without building those layers themselves.
