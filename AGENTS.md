# AGENTS.md — universal AI agent instructions for aenix.io

**This file is the universal counterpart to `CLAUDE.md`. It binds any AI assistant (Cursor, GitHub Copilot, Sourcegraph Cody, JetBrains AI, Continue.dev, etc.) working on this repository.**

If you are an AI assistant: read this file before any action on the repo.

## Single rule

**Read `CLAUDE.md` and follow it.**

`CLAUDE.md` contains the complete set of rules for this project — content rules, SEO/GEO requirements, mandatory references, page creation workflow, and conflict-resolution guidance. The same rules apply regardless of which AI assistant is being used.

## Why this file exists separately

Different AI assistants look for different filenames:
- Claude Code looks for `CLAUDE.md`
- Cursor looks for `.cursorrules` and `AGENTS.md`
- Many open-source AI tools look for `AGENTS.md` as a shared standard

Rather than maintain N files with N copies, this `AGENTS.md` and `.cursorrules` both point to `CLAUDE.md` as the source of truth.

## If you're a human reading this

`CLAUDE.md` has the rules. Follow them too.
